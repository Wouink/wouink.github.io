Installing

  • With MultiMC
    1. Download the modpack Zip file (link above)
    2. In MultiMC: Right Click > Create Instance
    3. In the left-hand side menu, select Import from Zip
    4. Ok
  • With Minecraft Launcher
    1. Download the modpack Zip file (link above)
    2. Follow my tutorial (in french...) to import a modpack
